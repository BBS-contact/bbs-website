About the SOS Module

The SOS Module is a critical component of the BBS system, designed to allow people in danger or distress to immediately call for help — contacting police, emergency services, or family members in real time. The system works via mobile devices, includes GPS-based location tracking, pre-configured alert messages, and special accessibility features for users with disabilities.
